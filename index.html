Mission Activation and Emergency Response Intelligence

Proof of Consent for Toll-Free SMS Verification

Application: Brav Buddies Emergency Mobilization System

1. User Registration & Phone Number Collection

Users (firefighters, soldiers, and emergency personnel) register through a secure system where their phone numbers are collected for emergency communication purposes.

2. Message Types

  • Incident activation alerts
  • Personnel availability requests
  • Training reminders

3. Opt-In Process (Web Form Consent)

Users provide explicit written consent through a secure web-based form prior to receiving any SMS messages.

Consent Language:

By providing your mobile number and checking this box, you agree to receive SMS alerts from Brav Buddies Emergency Mobilization System about incident activations, availability checks, and training alerts.

Message frequency varies. Message and data rates may apply.

Reply HELP for help, STOP to unsubscribe at any time. Carriers are not liable for delayed or undelivered messages.

SMS opt-in data and consent will not be shared with third parties.

Consent Capture Requirements:

  • Consent checkbox is unchecked by default
  • User must actively check the box before submission
  • Consent is a separate action and not bundled

Record Keeping:

  • Phone number, timestamp, and consent status stored
  • IP address and submission logs recorded
  • Consent logs retrievable for verification

Confirmation Message:

You have opted in to receive emergency alerts from Brav Buddies Emergency Mobilization System.

Reply STOP to opt out, HELP for assistance. Msg & data rates may apply.

4. Sample Message Content

INCIDENT ALERT: Structure Fire
Location: 123 Main St
Report Time: 14:30
Reply AVAILABLE or UNAVAILABLE

- Emergency Mobilization System
Reply STOP to opt out, HELP for assistance. Msg & data rates may apply.

5. Opt-Out Mechanism

  • Reply STOP to unsubscribe
  • Reply HELP for assistance
  • Contact organization leadership

6. Data Privacy & Security

  • Data stored securely
  • Access limited to authorized personnel
  • No third-party sharing of consent data

7. Use Case Justification

This system is used for emergency response coordination. Messages are time-sensitive and operational, not promotional.

Declaration: All users provide explicit consent prior to receiving SMS communications.
